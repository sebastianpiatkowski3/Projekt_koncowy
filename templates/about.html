Aplikacja jest zbudowana na frameworku Flask i korzysta z API FreeCurrencyAPI do pobierania najnowszych kursów walut. Aplikacja składa się z trzech głównych stron:

  • Strona główna: Wyświetla najnowsze kursy walut. Kursy są pobierane za pomocą metody `latest()` z klienta FreeCurrencyAPI.
  • Status: Wyświetla status klienta API, w tym identyfikator konta i informacje o wykorzystaniu kwoty. Status jest pobierany za pomocą metody `status()` z klienta FreeCurrencyAPI.
  • O nas: Wyświetla informacje o naszej aplikacji.

Klucz API do FreeCurrencyAPI jest wczytywany za pomocą klasy Reader, która wczytuje klucz z pliku. Klucz ten jest następnie używany do inicjalizacji klienta API.
